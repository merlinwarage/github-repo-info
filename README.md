# Github Repository Info

## Live demo
http://mwserv.sytes.net/github-repo-info/

## Install/Run

`npm install` 

`ng serve` 

Navigate to `http://localhost:4200/`

## Running unit tests

Run `ng test` to execute the unit tests.

## Running end-to-end tests

Run `ng e2e` to execute the end-to-end tests.
